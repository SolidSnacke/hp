Теллеграмм-бот опросник проверит ваши знания по советским фильмам.

Вы будите должны ответить на 11 вопросов.

Ответы можно писать в виде текста или же просто можно написать номер ответа.
В конце опроса произойдет оценка ваших знаний.

Прошу пройти опрос до конца.



Команды для управленя:

/start - начало опроса

/stop - предварительная остановка опроса без показа результата.

/easter_egg - [ДАННЫЕ УДАЛЕНЫ]
